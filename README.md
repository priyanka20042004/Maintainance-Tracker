Maintenance Tracker System:-

The Maintenance Tracker System is a simple and efficient Java-based web application that helps users raise complaints, register machines, track maintenance work, and manage resolution status.  
This project is built using Java Servlets, JSP, HTML/CSS, and NetBeans, demonstrating backend logic, form handling, and basic CRUD-like operations.

Features:-

1.User Module
- Raise a complaint  
- Upload proof (image / document)  
- View complaint status  
- Track updates  

2.Technician Module
- View assigned complaints  
- Mark problem as resolved  
- Update service details  

3. Admin Module
- Register machines  
- Assign complaints to technicians  
- View all complaints  
- Close complaints

 How to Run the Project:-

1. Open NetBeans  
2. Go to File → Open Project 
3. Select this folder  
4. Right-click project → Clean and Build
5. Run using `F6`
